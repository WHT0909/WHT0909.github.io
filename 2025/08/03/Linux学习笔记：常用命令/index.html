(1) Linux 文件目录管理

常用的文件管理指令

指令 作用
ls -l 和 ll 用于显示文件属性，ls -l 是 ll 的简略版
cd 切换目录
ls -al 查看当前路径下的所有文件
pwd 查看当前所在目录
mkdir 文件夹名 新建文件夹
tree 以树形方式展示当前目录结构
touch 文件名 新建文件
ln -s 文件名 链接名 建立链接
chmod +w/r/x 文件名 修改文件权限
chown 所属用户:属组 _fileName 修改文件所属用户及属组
find 目录 -name 文件名 找文件
cp 原文件名 复制的文件名 复制文件
scp [选项] [源文件] [目标路径] linux 服务器之间传文件
mv 文件名 目录 移动文件
rm 文件名 删除文件
rm -r 目录 递归删除目录里的文件

部分命令演示

图 1：ls -l 命令

其中 w 表示可写， r 表示可读， x 表示可执行， - 表示没有对应的权限

第一个位置如果是 - 就代表这是目录文件。剩下的每三个字符为一组，分别表示属主权限、属组权限、其他用户权限。属主权限和属组权限分别对应 ls -l 命令输出的第 3、4 列，其他用户权限就是既不是属主，也不属于属组的所有其他用户（包括系统上的其他账户）拥有的权限

图 2：ln -s _fileName _linkName 命令建立链接

图 3：chmod +_ _fileName 命令修改文件权限

chmod a+x 1.log 表示为所有用户增加 1.log 的执行权限

权限也可以用数字表示：读为 4 ，写为 2 ，执行为 1 ；例如 chmod 777 1.log表示所有用户现在都有了对文件的读、写、执行权限

图 4：用数字表示权限

其他知识点

.表示当前目录，..表示上级目录

/home 下存放了不同的用户文件夹，可用adduser _userName添加用户，用deluser _userName删除

(2) Linux 文件查看编辑

指令 作用
cat 文件名 查看文件内容
head -n 行数 文件名 只看文件的前几行
tail -n 行数 文件名 只看文件的后几行
tail -f 文件名 查看文件末尾，滚动输出
more 文件名 显示文件内容，只展示第一屏
less 文件名 显示文件内容，只展示第一屏；按向上 / 向下箭头实现翻页
vim 文件名 编辑文件

vim 指令

默认打开文件时是只读模式，按上下箭头索引行

按 i 进入插入模式，可编辑文件

按 esc 切换成只读模式

只读模式下用:wq退出文件

只读模式下用:set nu显示行号

用:行号跳转到某一行

:q：如果文件没有任何修改，可以使用这个命令退出 Vim

:q!：如果文件有修改但不想保存，可以使用这个命令强制退出 Vim

:e!：放弃所有修改，从上次保存文件开始重新编辑
